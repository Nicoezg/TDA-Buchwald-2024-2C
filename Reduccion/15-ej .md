Se tiene una matriz donde en cada celda hay submarinos, o no, y se quiere poner faros para iluminarlos a todos. El problema es dar la cantidad mínima de faros que se necesitan para que todos los submarinos queden iluminados, siendo que cada faro ilumina su celda y además todas las adyacentes (incluyendo las diagonales), y las directamente adyacentes a estas (es decir, un “radio de 2 celdas”). ¿Se encuentra este problema en NP? ¿Qué problema NP-Completo visto en la cursada es parecido al problema definido? Definir en ambos casos el problema de decisión. ¿qué reducción podríamos hacer? ¿Podemos concluir que este problema es un problema NP-Completo?

Para definir si el problema se encuentra en NP, es necesario implementar un verificador en tiempo polinomial.

En clase se vio el problema NP-Completo set cover en donde tenemos un set U de n elementos y un listado s1, s2, s3..., sm de subsets de U. Un Set Cover es una selección de estos Subsets tal que la unión es U.

Defino en ambos casos el problema de decisión:

Problema de decisión submarinos: Dada una matriz nxn donde en cada celda hay submarinos o no, se quiere ilumar a todos poniendo faros. Estos faros iluminan en un radio de 2 celdas. Es posible iluminar todos los submarinos con a lo sumo k faros?

Problema de decisión set cover: Dado un set U de n elementos y un listado s1, s2, s3..., sm de subsets de U, un set cover es una seleccion de estos subsets tal que la union es U. Existe un subset cover de a lo sumo K subsets?

Podemos reducir el problema de los submarinos a set cover. Para ello, es necesario transformar el problema. Pensaremos s1, s2, s3 ..., sm como los distintos faros posibles, donde cada uno de estos contendrá a los submarinos a los cuales ilumina. Podemos transformar el set U de n elementos a un set S de m submarinos. Entonces, podremos utilizar la resolución del problema de set cover para resolver el problema de los submarinos. Tendremos una cantidad m de submarinos a los cuales queremos iluminar y tendremos faros los cuales iluminan a distintos submarinos de ese conjunto S. Buscaremos si podemos iluminar con distintos valores de K, hasta que K - 1 nos señale que es imposible iluminar a todos los submarinos. Una vez transcurra esto, sabremos la minima cantidad de faros que debemos utilizar (K).

Si existe una cantidad k de faros, existe una cantidad k de subsets para el problema transformado y si hay una cantidad k de subsets hay una cantidad k de faros.

Si al problema original le ponemos k faros, esos k faros cubrieron todos los submarinos. En el problema transformado si seleccionamos esos mismos faros, estan representados por subsets, entonces, para cualquier submarino en el problema original hay un faro que lo ilumina. Por lo tanto en el problema transformado existe el Si tal que Si activa al Ux para cualquier Ux. Como la cantidad de faros es k, la cantidad de subsets es K.

Existen k subsets cubren todos los elementos. Eso significa que para cualquier submarino de Sx ese Sx tiene un Ux que queda cubierto con Si entonces selecciono ese subset Si, agarro el faro correspondiente, agarro una cantidad de k faros y por lo tanto existen k faros que cubren los submarinos.
